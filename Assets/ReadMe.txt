EasyShortCutLockInspector 

[OverView]
EasyShortCutLockInspector Is a Shortcut  Lock Inspector key to toggle Inspector option unlock/lock.

[System Requirement]
In all Unity Editor

[How to use and detailed description]

Add the Script in your Editor folder; in this it will work even without opening any scene.
After adding, hit “command+L or CTRL+L” for changing option unlock/lock option.
